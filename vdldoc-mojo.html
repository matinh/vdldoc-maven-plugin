vdldoc:vdldoc

Full name:

com.github.matinh.vdldoc:vdldoc-maven-plugin:1.0-SNAPSHOT:vdldoc

Description:

Generate documentation for JSF tag libraries via OmniFaces Vdldoc.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: site.

Optional Parameters

Name Type Since Description
browserTitle String - Browser window title.
Default value is: ${project.name} VDL Documentation.
documentTitle String - Documentation title.
Default value is: ${project.name} VDL Documentation.
excludes List - Patterns to exclude when searching for taglib descriptor files.
Default value is: target/**.
User property is: excludes.
failOnError boolean - If false, build will continue when generation of documentation fails.
Default value is: true.
User property is: maven.vdldoc.failOnError.
includes List - Patterns to include when searching for taglib descriptor files.
Default value is: **/*.taglib.xml.
User property is: includes.
outputDirectory File - Location of the output directory for the generated documentation.
Default value is: ${project.build.directory}/vdldoc.
User property is: maven.vdldoc.outputDirectory.
skip boolean - Skip the generation of VDL documentation.
Default value is: false.
User property is: maven.vdldoc.skip.

Parameter Details

browserTitle:

Browser window title.
  • Type: java.lang.String
  • Required: No
  • Default: ${project.name} VDL Documentation

documentTitle:

Documentation title.
  • Type: java.lang.String
  • Required: No
  • Default: ${project.name} VDL Documentation

excludes:

Patterns to exclude when searching for taglib descriptor files.
  • Type: java.util.List
  • Required: No
  • User Property: excludes
  • Default: target/**

failOnError:

If false, build will continue when generation of documentation fails.
  • Type: boolean
  • Required: No
  • User Property: maven.vdldoc.failOnError
  • Default: true

includes:

Patterns to include when searching for taglib descriptor files.
  • Type: java.util.List
  • Required: No
  • User Property: includes
  • Default: **/*.taglib.xml

outputDirectory:

Location of the output directory for the generated documentation.
  • Type: java.io.File
  • Required: No
  • User Property: maven.vdldoc.outputDirectory
  • Default: ${project.build.directory}/vdldoc

skip:

Skip the generation of VDL documentation.
  • Type: boolean
  • Required: No
  • User Property: maven.vdldoc.skip
  • Default: false
